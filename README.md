DoConnect is a modern Q&A platform where experts and learners connect. Ask questions, share knowledge, and grow together.

### Stack

- **Vite**
- **TypeScript**
- **React**
- **shadcn-ui**
- **Tailwind CSS**

### Getting started

- **Install dependencies**

```sh
npm install
```

- **Run the dev server**

```sh
npm run dev
```

The app will start on the port configured in `vite.config.ts` (by default, 8080).

### Build for production

```sh
npm run build
```

### Preview production build

```sh
npm run preview
```
